## 🔍 Overview

Ultron Frontend is a **React 19** single-page application that provides a polished, mobile-friendly chat interface for the Ultron AI assistant. It communicates with the backend in real time via **Socket.IO**, renders animated message bubbles using **Framer Motion**, and supports category-based FAQ suggestions for guided conversations.

---

## ✨ Features

- 💬 **Real-time chat** via Socket.IO with WebSocket-first transport
- 🎞️ **Animated message bubbles** — each response splits into multiple bubbles with staggered entrance animations
- 🗂️ **Category-based suggestions** — browse FAQ topics and get rotating question pairs
- 🔄 **Scrollable category carousel** with drag-to-scroll on desktop
- ⌨️ **Typing indicator** with animated, color-coded bouncing dots
- 👍 **Feedback widget** — thumbs up/down after all suggestions are exhausted
- 📱 **Mobile-optimized** — respects `visualViewport` height for correct layout when the keyboard opens
- 🎬 **Animated welcome screen** with looping Ultron video and gradient headline
- ♻️ **Chat session management** — create new chats and restore sessions seamlessly
- 🔁 **Auto-reconnect** — Socket.IO configured with retry logic (5 attempts, 1s delay)
- 🧭 **File-based routing** via TanStack Router — `/` redirects to `/chat`

## 🚀 Getting Started

### Prerequisites

- **Node.js** v18 or higher
- **npm**, **yarn**, **pnpm**, or **bun**
- A running instance of the [Ultron Backend](../Ultron_Backend/)

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/Ultron_Frontend.git
cd Ultron_Frontend

# Install dependencies
npm install
```

### Environment / Config

The backend URL is configured directly in two source files. Update these to match your backend:

**`src/lib/api.ts`** — Axios base URL:
```ts
const baseAPI = axios.create({
  baseURL: "http://localhost:5000",  // ← change to your backend URL
  withCredentials: true,
});
```

**`src/lib/socket.ts`** — Socket.IO connection URL:
```ts
export const socket = io("http://localhost:5000", {  // ← change to your backend URL
  autoConnect: true,
  withCredentials: true,
  transports: ["websocket", "polling"],
});
```

> 💡 **Tip:** You can optionally extract these into a `.env` file using `import.meta.env.VITE_BACKEND_URL` for easier environment switching.

### Running the App

```bash
# Development server (port 3000)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run tests
npm test
```

App will be available at `http://localhost:3000`.

## 📖 Usage Guide

### Starting a Conversation

1. **Open the app** — you're greeted with the Ultron welcome screen
2. **Choose a category** from the scrollable pill carousel (e.g. "Get Started", "About Me")
3. **Click a suggested question** or type your own message in the input bar
4. **Ultron responds** in real time with animated message bubbles

### Suggested Questions

- After selecting a category, two FAQ questions are shown at a time
- Each sent question advances the pair to the next two suggestions
- Once all suggestions are seen, a **"Was this helpful?"** feedback prompt appears
- Clicking 👍 or 👎 logs feedback and resets the category to the main list

### New Chat

- The new chat action clears the current session and resets all state
- Previous chats can be restored by emitting the `chat_history` socket event from the backend

## 🔌 Socket Integration

The app uses a centralized socket instance (`src/lib/socket.ts`) shared across all components.

### Emitted Events

| Event | Payload | When |
|---|---|---|
| `send_message` | `{ msg: string, chatId: string \| null }` | User sends a message |
| `get_suggestions` | `{ category: string }` | User selects a FAQ category |

### Listened Events

| Event | Payload | When |
|---|---|---|
| `receive_message` | `{ chatId, question, answer }` | AI responds to a message |
| `chat_history` | `{ chatId, messages[] }` | Restoring a past session |
| `suggestions` | `{ questions: string[] }` | FAQ suggestions loaded |
| `error_message` | — | Server-side error occurred |

All listeners are registered in `src/services/chatSocket.ts` and cleaned up on component unmount.

## 🎨 UI & Design

### Color Palette

| Usage | Value |
|---|---|
| Background | `#000000` (pure black) |
| Surface | `rgba(255,255,255,0.05)` |
| Border | `rgba(255,255,255,0.10)` |
| Text Primary | `slate-200` |
| Text Muted | `slate-500` |
| Accent (Send button) | `#ff0000` (red) |
| Gradient | Conic: `#1a1a1a → #4a4a4a → #ff0000 → #b0b0b0` |

### Message Bubbles

AI responses are automatically split into multiple bubbles using the `splitIntoBubbles()` utility:

1. Split by double newlines (paragraphs)
2. Fall back to single newlines
3. Fall back to sentence-boundary splitting (max 130 chars per bubble)

Each bubble animates in with a staggered delay via Framer Motion (`AnimatePresence`).

### Typing Indicator

Three dots animate with bouncing motion, each colored differently (`#4a4a4a`, `#ff0000`, `#b0b0b0`) to match the Ultron conic gradient theme.
